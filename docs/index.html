About the project

PHGEODEX provides an interactive interface for exploring geospatial indicators across multiple administrative levels in the Philippines.

The platform allows users to navigate from national and regional scales to provinces, municipalities, cities, and barangays. Indicators can be explored spatially through interactive maps, classification schemes, and administrative filtering.
