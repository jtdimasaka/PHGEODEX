Data

PHGEODEX combines geographic boundaries and indicator datasets to support multi-scale spatial analysis across the Philippines.

Administrative boundaries

The platform supports multiple administrative levels:

  • Regions
  • Provinces
  • Municipalities and cities
  • Barangays

Geospatial data format

Vector geographic data are distributed using vector tiles to enable efficient visualisation of large geographic datasets in the browser.
