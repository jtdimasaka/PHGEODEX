Methodology

This page describes the methodological framework underlying PHGEODEX.

1. Geographic framework

The platform organises information across multiple administrative levels, including regions, provinces, municipalities, cities, and barangays.

2. Indicator construction

Indicators are calculated from geospatial and administrative datasets and represent the distribution of selected characteristics across geographic units.

3. Spatial aggregation

Values are aggregated across administrative units to allow users to examine spatial patterns at different geographic scales.

4. Classification

The map uses data-driven classification methods to divide indicator values into interpretable classes for visualisation.

5. Interactive visualisation

Users can select administrative units and indicators to dynamically update the map. The visualisation is designed to support exploration from national-scale patterns to local-level geographic variation.

Note: Detailed methodological information should be added here as the underlying research and indicator definitions are finalised.
